# orc-cli-hello-world

## Description
A POC for orc programming language.

## Tech stack
- java

## Docker stack
- openjdk:8
- rzuckerm/orc

## To run
`sudo ./install.sh -u`

## To stop (optional)
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
